# 股票投资分析自查手册（逻辑修订版 v2.0）

> **目的**：每一笔交易前，必须按此框架写下答案，避免随机交易、冲动交易和被套后临时改口径。  
> **原则**：先确定交易周期，再确定买入理由、证伪条件、仓位和退出规则。  
> **风险提示**：本手册只用于个人投资决策自查，不构成任何投资建议；任何交易都可能亏损。

---

## 前置快速否决清单（30 秒过滤）

> 盘中来不及完整走流程时，先过这一关。触发任一条，暂时放弃，等收盘后有充足时间再对照手册详细研究。

- [ ] 大股东/实控人近 3 个月有大幅减持，且无合理解释（合理解释举例：个人税务规划、偿还股权质押贷款、协议转让给战略投资者且仍为实控人；不合理：清仓式减持、业绩变脸前突击减持、多次窗口期踩线操作）
- [ ] 最近两个会计年度扣非净利润均为负
- [ ] 被监管立案调查，或审计报告为非标意见
- [ ] 近一个月股价暴涨超过 80%，且没有对应的基本面改善（不追高）

> **例外：** 若本次明确按“困境反转股”研究，则**不适用第 2 条（扣非为负）**，必须走完整流程（含步骤四困境反转估值法与步骤六反转逻辑）。该条只用于过滤盘中对普通股的冲动追买。

## 步骤一：界定投资周期（先锁定这一笔的性质）

| 维度 | 短线/波段（<3个月） | 中线（1-2年） | 长线（>3年） |
|------|---------------------|---------------|--------------|
| 本次交易属于 | □ | □ | □ |

**匹配检验（必须直接回答，不允许只贴标签）：**

- 若选**短线/波段**：未来 1 个月内是否有明确催化剂？例如财报、政策、订单、产品发布、行业价格变化。  
  → 无明确催化剂，短线逻辑不成立。
- 若选**中线**：未来 1-2 年利润或自由现金流能否明显改善？改善来自哪里？产品放量、成本下降、周期反转、价格上涨、费用率下降，必须至少写出一个量化假设。  
  → 若无法给出精确量化，至少给出区间假设及跟踪指标（如“核心客户数同比增长 20%-30%”“费用率降至 25%-28%”）。  
  → 完全无法量化且无跟踪指标，中线逻辑不成立。
- 若选**长线**：公司是否具备长期复利基础？重点检查：商业模式稳定、过去 5 年盈利质量较好、经营现金流可靠、行业竞争格局没有恶化。  
  → 长线不是“拿得久”，而是“逻辑经得起长期验证”。

**禁止交叉逻辑：** 不要因为短线破位去卖长线仓，也不要用“护城河”“长期看好”为短线被套找理由。每一笔交易只能按入场时设定的周期和规则处理。

### 2.2 硬性否决项（出现任一条，原则上停止分析）

- [ ] 扣非净利润连续 2 年低于净利润的 50%，且没有合理解释。
- [ ] 年归母净利润 < 5000 万，且主营业务缺少稳定增长证据。
- [ ] 经营现金流/净利润连续 3 年 < 0.5，说明利润质量偏弱。
- [ ] 应收账款或存货连续明显上升，但收入和现金流没有同步改善。
- [ ] 商誉占净资产比例过高，且存在并购标的业绩下滑风险。
- [ ] 监管处罚、财务造假、审计保留意见、实控人风险等重大负面事项。

> **困境反转股例外：** 本组否决项中“盈利质量/利润规模”类条款（前两条）针对正常经营公司；困境反转股需结合步骤四反转逻辑单独判断，不直接套用。

### 2.3 长线质量加分项（不是硬门槛，但越多越好）

- [ ] 连续 5 年 ROE > 15%，且不是靠高杠杆堆出来。
- [ ] 经营现金流/净利润长期 > 0.8。
- [ ] 毛利率和净利率稳定，未出现持续恶化。
- [ ] 分红稳定，且不影响正常经营投入。
- [ ] 行业格局清晰，公司具备成本、品牌、渠道、技术或资源优势。

## 步骤三：区分行业类型（普通股、成长股、周期股不能混用公式）

| 类型 | 特征 | 主要看什么 | 常见错误 |
|------|------|------------|----------|
| 稳定价值股 | 盈利稳定、分红稳定、估值波动不大 | 现金流、分红、估值分位 | 只看低 PE，忽略增长停滞 |
| 成长股 | 收入和利润仍在扩张 | 增速、空间、竞争格局、费用率 | 用静态 PE 判断贵贱 |
| 周期股 | 利润受价格、库存、供需强烈影响 | 产品价格、库存、产能、行业亏损比例 | 在景气顶点买低 PE |
| 困境反转股 | 过去差，未来可能改善 | 亏损收窄、资产处置、订单、管理层变化 | 把“便宜”误认为“反转” |

**先判断类型，再选择估值方法。类型判断不清，不买。**

> **估值方法对应表：** 稳定价值股 / 成长股 → 4.1 公式；周期股 → 4.3 框架；困境反转股 → 4.4 反转估值法。

## 步骤四：回报率测算——用数字判断贵还是便宜

### 4.1 普通非周期股公式

> 3 年预期年化回报率 ≈ ((未来第 3 年 EPS / 当前 EPS) × (目标 PE / 当前 PE))^(1/3) - 1 + 当前股息率

**参数取值：**

- **当前 EPS**：最近 12 个月每股收益，或最近一年扣非 EPS。
- **未来第 3 年 EPS**：根据收入增速、利润率、股本变化估算，必须写出假设。
- **当前 PE**：PE-TTM，周期股和亏损股慎用。
- **目标 PE**：优先使用公司近 5 年 PE 中位数；若上市不足 5 年或 PE 长期异常，使用行业合理估值区间。
- **当前股息率**：最近 12 个月每股分红 ÷ 当前股价。

**举例：**  
当前 EPS=1 元，预计 3 年后 EPS=1.3 元，当前 PE=15，目标 PE=12，股息率=2%。  
3 年预期年化 ≈ ((1.3/1.0) × (12/15))^(1/3) - 1 + 2% ≈ 3.3%。

**风险提示与压力测试：**

- 未来 EPS 仅为假设，若实际利润与假设偏离超过 20%，应重新评估。
- 建议追加 **零增长压力测试**：假设未来利润零增长，仅靠估值修复和股息，预期年化回报是多少？若该数字仍高于门槛，安全边际才真正够厚。

### 4.2 买入门槛

- 普通股票：预期年化回报率建议 **≥ 8%**。
- 高波动、小盘、强周期、困境反转：预期年化回报率建议 **≥ 12%**。
- 如果只达到“10 年期国债收益率 × 2”，通常不足以覆盖股票波动、判断错误和流动性风险。
- 10 年期国债收益率只作为无风险收益参考，填写时以当天实际数据为准，不写固定值。

### 4.3 周期股专用检查（不使用普通 PE 回归公式）

| 检查项 | 当前情况 | 结论 |
|--------|----------|------|
| 核心产品价格处于周期低位/中位/高位？ | | |
| 行业库存是上升还是下降？ | | |
| 行业新增产能是否集中释放？ | | |
| 行业是否已有大量企业亏损？ | | |
| 公司成本是否显著低于同行？ | | |
| 当前利润是否处于历史高位？ | | |
| 当前 PB 是否处于历史低位（建议近 10 年 PB 分位 ≤ 30%）？ | | |

**周期股量化门槛（满足才进入研究）：** PB 处于近 10 年 **30% 分位以下**，且当前利润/ROE 不处于历史高位。  
**前提条件：** 公司近 10 年主营业务未发生根本性变化；若发生过重大资产重组，改用可比公司当前 PB 作参考。  
**周期股买入原则：** 优先在“行业亏损、库存下降、产能出清、价格有企稳迹象”时研究；警惕在“利润历史高位、ROE 和毛利率历史高位、PE 很低”时买入。

### 4.4 困境反转股专用估值（当前 EPS 为负或异常，不能直接套 4.1）

> 困境反转股当前盈利往往为负或接近零，PE 公式失效。改用**正常化盈利**估算，并对反转的兑现设硬性时间表。

**估值步骤：**

1. **估算正常化 EPS（Normalized EPS）**：假设公司恢复到行业正常利润率（取同行近 5 年中位净利率，或公司历史正常年份净利率），用预期收入 × 正常化净利率 ÷ 总股本，得到反转兑现后的 EPS。必须写出收入和利润率假设。
2. **套用 4.1 公式**：把“未来第 3 年 EPS”替换为正常化 EPS，目标 PE 用行业正常估值中位数，算出 3 年预期年化回报率。
3. **门槛**：预期年化回报率 **≥ 12%**（高不确定性，要求更高安全边际）。

**反转兑现时间表（强制写出，否则不买）：**

- 反转最晚应在哪个季度/年度体现？用什么财务或经营数据验证（如“连续两个季度扭亏”“产能利用率回到 70% 以上”“核心产品价格站上成本线”）？
- 反转证伪信号（具体到数字或事件）：到期未兑现、亏损继续扩大、资产进一步减值等。

> **关键提醒：** “便宜”不等于“反转”。没有明确的反转驱动力和验证时间表，只是低估值的差公司，按否决处理。

## 步骤五：筹码与博弈——只用于排雷，不用于发现机会

| 信号 | 检查方法 | 结论 | 处理动作 |
|------|----------|------|----------|
| 大股东/实控人近 3 个月明显减持 | 公告，关注比例、原因、价格 | 减持比例大且无合理解释 → 🔴 | 否决，除非步骤六能写出清晰证伪逻辑。合理解释参考：个人税务规划、偿还股权质押贷款、协议转让给战略投资者且仍为实控人；不合理：清仓式减持、业绩变脸前突击减持、多次窗口期踩线操作 |
| 股东户数本季暴增 >30% | 季报对比 | 筹码明显分散 → 🔴 | 否决，除非步骤六能写出清晰证伪逻辑 |
| 融资余额连续上升而股价滞涨 | 近 20 日数据 | 杠杆资金拥挤 → 🟠 | 不否决，但下调计划仓位，避开杠杆退潮期 |
| 北向资金持续净流出 | 近 20 日数据 | 对外资定价权强的股票更重要 → 🟠 | 不否决，但下调计划仓位，列入观察 |
| 未来 1 个月有大量解禁 | 解禁日程 | 不必然下跌，但会压制预期 → 🟡 | 不否决，可推迟买点至解禁落地后 |
| 高管或公司主动回购 | 公告 | 加分，但要看金额和真实执行 | 加分项，不构成买入理由 |

**否决规则：** 任何🔴信号出现，且无法在步骤六写出清晰证伪逻辑，则直接否决。  
**注意：** 筹码信号不是买入理由，只能作为风险过滤器。减持的硬伤判断统一在本步骤处理，步骤六事件快筛不再重复打分。

## 步骤六：故事与逻辑——凭什么是你赚钱？

### 6.1 事件快筛（先扫硬伤）

> 大股东/实控人减持已在步骤五统一打分与否决，此处不再重复。

| 检查项 | 是/否 | 说明 |
|--------|-------|------|
| 公司近期有回购计划？ | | 是→看金额、价格上限和执行情况 |
| 近 3 个月有机构研报覆盖？ | | 否→不直接否决，但需补足信息来源 |
| 有负面舆情、监管函、立案？ | | 是→重大负面直接否决 |
| 有股权激励方案？ | | 是→考核目标是否足够高、能否实现 |
| 行业政策是否有重大变化？ | | 是→判断利好还是利空 |

### 6.2 三段式逻辑（强制填写，缺一不可）

1. **我买的核心逻辑**：市场对它有什么误解？它和表面数字哪里不同？我的优势信息或判断是什么？
2. **证明我对的催化剂**：最晚在什么时间、什么数据或事件会验证逻辑？
3. **我错了的信号**：必须具体到数字或事件，例如“下一季度营收同比下滑超 5%”“大股东公告大额减持”“核心产品价格跌破 XX”“跌破 XX 元且无法收回”。

**若第 3 点写不出，禁止买入。因为这说明你不知道什么时候该认错。**

## 步骤七：技术面——只做趋势确认，不替代基本面

| 周期 | 观察指标 | 判断标准 |
|------|----------|----------|
| 短线/波段 | 20 日均线 | 股价在线上且均线向上，为可操作区；跌破后谨慎 |
| 中线 | 60 日均线 | 股价在线上且均线向上，辅以周线 MACD 或平台突破确认 |
| 长线 | 20 月均线 | 月线长期位于其下且均线走平向下，不加仓 |

**成交量辅助：** 上涨温和放量、下跌缩量较健康；放量暴跌、无量阴跌、破位后反抽无量，均视为趋势变弱。  
**核心原则：** 技术面只负责“买点和风险确认”，不负责“证明公司值得买”。

## 步骤八：决策与仓位管理

### 8.1 买入前最终确认（打勾）

- [ ] 投资周期已选定，并回答了对应匹配检验。
- [ ] 公司类型已判断清楚：稳定价值股/成长股/周期股/困境反转股。
- [ ] 财务无硬性否决项。
- [ ] 回报率测算达到对应门槛：普通股 ≥ 8%，高波动/周期/反转 ≥ 12%。
- [ ] 筹码面没有🔴信号，或已写出清晰证伪逻辑。
- [ ] 三段式逻辑已写，证伪点具体可执行。
- [ ] 买入价、止损价、逻辑证伪点、计划仓位均已记录。
- [ ] 这笔交易即使失败，也不会影响正常生活和心态。

### 8.2 仓位公式

> 单票仓位上限 = 单笔可承受账户亏损 ÷ 止损幅度

**举例：**  
账户最多允许单笔亏损 2%，本次止损幅度 15%。  
单票仓位上限 = 2% ÷ 15% = 13.3%。  
即使你很看好，也不应超过这个仓位太多。

**取小值原则：** 本公式算出的“风险仓位上限”与下方“单票 ≤ 30%”绝对上限，**两者取较小值**作为最终仓位。

**仓位规则：**

- 新开仓先用计划仓位的 1/2，逻辑验证后再加。
- 单票原则上 ≤ 30%，除非是极高确定性的低波动品种。
- 单一行业仓位原则上 ≤ 50%。
- **现金仓位**：视资金量灵活处理，但不可长期满仓。
  - 资金量较小（< 10 万）：现金比例不作硬性要求，但必须始终保留足以应急离场的流动性，且不因摊低成本而动用全部资金；
  - 资金量较大（≥ 50 万）：建议现金或等价物 ≥ 20%。
- 高波动股票和周期股仓位要自动下调。

| | | | | | | □是 □否 | □是 □否 | |

**每月三问：**

1. 买入逻辑还在吗？证伪点是否已经触发？
2. 有性价比更高、风险更低的替代标的吗？
3. 行业集中度、单票仓位、现金比例是否仍在规则内？（小资金账户现金比例不作硬性要求，但需保留应急离场的流动性）

**处理规则：**

- 加仓只能发生在逻辑被验证后，而不是因为亏损后想摊低成本。

## 步骤十：卖出执行

### 10.1 止损

- **短线/波段**：价格止损优先。触发止损线，按计划离场，不幻想。
- **中线/长线**：逻辑证伪优先，价格止损作为风险底线。若价格跌破止损但逻辑未变，必须重新评估仓位是否过重。
- **盘后记录**：这笔亏损教给我什么？是买点错、逻辑错、仓位错，还是执行错？

### 10.2 止盈/逻辑卖出

卖出前回答两个问题：

1. 当初逻辑是否已经充分兑现，或已经恶化？
2. 如果现在空仓，我是否还愿意以当前价格重新买入？

若任一答案为否，减仓或清仓。

### 10.3 暴涨主动止盈

- 一个月内涨幅超过 80%，且没有对应基本面改善，主动卖出 1/3。
- 剩余仓位用移动止盈或逻辑证伪点管理。
- 暴涨后不因“还会涨”加仓，只允许按计划处理。
